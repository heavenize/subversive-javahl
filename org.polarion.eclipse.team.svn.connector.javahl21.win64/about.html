About Subversive SVN 1.14 JavaHL Connector - Windows x64 Native Libraries

Version: 7.0.0
Copyright (c) 2005-2025 Polarion Software. All rights reserved.

This fragment bundle contains the native Windows x64 libraries required for the JavaHL 1.14 connector to function on Windows platforms.

Included Native Libraries

Apache Subversion 1.14.x

  • libsvn_client-1.dll
  • libsvn_delta-1.dll
  • libsvn_diff-1.dll
  • libsvn_fs-1.dll
  • libsvn_ra-1.dll
  • libsvn_repos-1.dll
  • libsvn_subr-1.dll
  • libsvn_wc-1.dll
  • libsvnjavahl-1.dll (JavaHL JNI binding)

Apache Portable Runtime (APR)

  • libapr-1.dll
  • libapriconv-1.dll
  • libaprutil-1.dll

OpenSSL 1.1.x

  • libcrypto-1_1-x64.dll
  • libssl-1_1-x64.dll

Microsoft Visual C++ Runtime 2015-2022

  • VCRUNTIME140.dll
  • MSVCP140.dll

Additional Dependencies

  • libsasl.dll (Simple Authentication and Security Layer)
